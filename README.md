# CampusConnect

A university social media + academic services platform.

## Roles
- Student
- Teacher
- Admin (hardcoded)

## Tech Stack
- Frontend: React + Tailwind CSS (Vite)
- Backend: Node.js + Express.js
- Database: MongoDB + Mongoose
- Auth: JWT + bcrypt
- File Storage: Local (Multer)

## Getting Started

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- Signup/Login (Student & Teacher), hardcoded Admin
- Home Feed, Post creation (text + image)
- Like/Unlike, Inline Comments
- Profile Pages
- 1-to-1 Messaging
- Club/Society Pages
- Notice Dropdown
- Academic Page (Routine, Result, Seat Plan PDFs)
- Admin Upload Panel
